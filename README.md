# Returns Experience Prototype

Protótipo de experiência de devoluções com seleção de artigos e múltiplos métodos de devolução.

## Funcionalidades

- **Seleção de artigos** — escolha quais artigos devolver com motivo obrigatório
- **Recolha em casa** — agendamento com escolha de slot horário
- **Ponto de entrega CTT** — localização dos pontos mais próximos com distância
- **Devolução em shopping** — lista de shoppings parceiros com distância e instruções
- **Confirmação e resumo** — revisão antes de submeter com valor de reembolso estimado

## Stack

- React 18
- Vite

## Instalação

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`

## Build

```bash
npm run build
```
